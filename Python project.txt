Local app, Coded with python, linked to an MS access database


This device has 3 tabs:
* 1 to create a spreadsheet to export as an excel with the desired columns and filters, 
* 1 to modify, add, delete rows from the database, 
* and 1 to import all the data to the database from different excel


when opening the device, a choice of profile will be asked: admin or users limiting our use of the device (see figma)
